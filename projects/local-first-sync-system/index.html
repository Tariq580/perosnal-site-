Local-first Sync System

An idea for syncing plain text files across devices without needing a traditional cloud account.

project link ↗

I like tools that stay simple and portable.

This idea keeps coming back: a local-first sync system for plain text files that feels lighter than a full cloud product. Something private, understandable, and boring in a good way.

I do not have the full shape of it yet, but the problem still feels worth returning to.
